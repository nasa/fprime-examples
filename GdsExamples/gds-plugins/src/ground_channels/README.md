# Ground-Derived Channels

This directory contains a simple plugin to derive the cosine from the sine telemetry. It is an example of the ground-derived (or calculated) channels. It consists of a [plugin](./cosine.py) and the associated [dictionary](./GroundDefinedChannelsDictionary.json) for the new channels.


This can be used with the [ExamplesDeployment](../../../../FlightExamples/ExamplesDeployment/) as long as the steps from the [README](../../../../FlightExamples/ExamplesDeployment/README.md#running-the-application-with-ground-calculated-channels) are followed.
